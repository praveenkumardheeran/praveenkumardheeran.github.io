## 📖 Overview

This portfolio represents my journey as a **Data Analyst**, **Python Automation Developer**, and **Frontend Developer**. It provides an overview of my experience, technical expertise, certifications, and featured projects while demonstrating my frontend development skills.

---

## ✨ Key Features

- Responsive design for desktop, tablet, and mobile
- Professional hero section with typing animation
- Interactive navigation with smooth scrolling
- Animated statistics section
- About, Experience & Skills sections
- Certifications showcase
- Project portfolio with GitHub links
- Resume download
- Contact section with social links
- Modern UI with custom animations

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/praveenkumardheeran/praveenkumardheeran.github.io.git
```

Navigate to the project folder

```bash
cd praveenkumardheeran.github.io
```

Open `index.html` directly in your browser or use the **Live Server** extension in Visual Studio Code.
